Cara Kerja Browser Dan Search Engine

Cara Kerja Browser

User sedang mengakses website dengan mengetikkan alamat pada situs maupun URL di address bar. Misal youtube.com. Lalu, web browser menerima request tersebut dan akan melakukan fetching atau pengambilan data pada DNS server. Data yang sudah di ambil tadi berupa IP dari request yang diketikkan user (www.youtube.com). Web browser telah mendapat IP dari www.youtube.com. Web browser mengakses server dengan IP yang sudah didapatkan dari DNS server. Server memberikan data konten dari www.youtube.com dalam bentuk HTML dan file lain seperti PHP dan CSS. Terakhir, web browser akan menampilkan konten yang sesuai dengan request user.

Cara Kerja Search Engine

Search engine bekerja dengan menjelajahi internet melalui program yang disebut "crawler" untuk mengumpulkan informasi dari situs web. Data yang dikumpulkan diindeks untuk pencarian selanjutnya. Ketika pengguna memasukkan kata kunci, mesin pencari mencocokkan permintaan pencarian dengan data yang ada dalam indeksnya dan menghasilkan hasil berdasarkan relevansi. Algoritma peringkat digunakan untuk menentukan urutan hasil pencarian yang paling relevan dengan kata kunci yang dimasukkan oleh pengguna.


Kelebihan Web Browser


  1. Akses yang mudah
  2. memudahkan pengguna mengakses informasi dan data secara cepat dan ringkas dari internet
  3. Kemudahan download
  4. website browser membantu pengguna mengunduh file atau data dari laman website ke komputer pribadi.

Kekurangan Web Browser

  1. Membutuhkan resource yang besar
  2. Saat baru membuka aplikasi maupun tab baru, prosessnya akan terasa sangat lambat.
  3. Kurang responsif dan tidak mendukung script tertentu.
  4. Rentan terhadap serangan malware virus, hingga hacking.
